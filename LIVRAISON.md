## ⚠️ Rappel important

**Ceci est un PROTOTYPE STATIQUE à des fins de démonstration uniquement.**

### Ne PAS utiliser en production tel quel
- Pas de vraie base de données
- Pas de backend sécurisé
- Pas de vrais paiements
- Données volatiles (localStorage)
- Sécurité non garantie

### Pour la production, il faut :
1. Backend Next.js + TypeScript
2. PostgreSQL + Prisma ORM
3. Auth JWT sécurisée
4. Intégration API paiement réelle
5. Upload de fichiers (S3/Cloudinary)
6. Tests automatisés
7. CI/CD pipeline
8. Monitoring et logs
